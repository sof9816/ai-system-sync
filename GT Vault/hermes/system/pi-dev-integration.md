## Adding a Skill

1. Create directory: `gt-core/skills-repo/<category>/<skill-name>/`
2. Write `SKILL.md` with YAML frontmatter (`name`, `description`, `category`, `tags`, `version`)
3. Validate: `validate-skills.py`
4. Commit in `gt-core/skills-repo/`
5. Sync: `sync-skills.py`

---

## Registering a Project

1. Create `.project.yaml` in project root (see template in `gt-core/templates/project-manifest.yaml`)
2. Validate: `validate-manifest.py .project.yaml`
3. Register via API or `project-detect.py . --register`

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Config API down | Start dashboard backend on port 7373 |
| Skills invalid | Run `validate-skills.py`, fix frontmatter |
| Provider switch failed | Check `$KIMI_API_KEY`, restore from `.bak` file |
| Sync not updating | `git pull` in `skills-repo`, then `sync-skills.py` |
| Cron not running | `setup-cron.py --install` |
